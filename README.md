# JitoX Dashboard

A secure and transparent dashboard for monitoring Solana MEV operations.

## Security Features

### End-to-End Encryption
- All communications between the dashboard and servers are encrypted
- Secure WebSocket connections for real-time data
- HTTPS-only access

### Authentication
- Secure Telegram login integration
- No phone number required
- Session-based authentication
- Automatic session expiration

### Wallet Security
- Read-only wallet connections
- No private key storage
- Real-time connection status monitoring
- Automatic disconnection on inactivity

## Transparency

### Open Source
- Full source code available on GitHub
- MIT License
- Community auditable
- Documented codebase

### Data Sources
- Direct integration with Solana blockchain
- Real-time data verification
- Transparent fee structure
- Verifiable transaction history

### Monitoring
- Real-time connection status
- Security indicator badges
- Update timestamps
- Performance metrics

## Getting Started

1. Visit [https://jitonexus.github.io/jitox-dashboard/](https://jitonexus.github.io/jitox-dashboard/)
2. Connect with Telegram
3. View your MEV statistics and performance

## Development

### Prerequisites
- Node.js 14+
- npm or yarn
- Solana Web3.js

### Installation
```bash
git clone https://github.com/JitoNexus/jitox-dashboard.git
cd jitox-dashboard
npm install
```

### Running Locally
```bash
npm start
```
